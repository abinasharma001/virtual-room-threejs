# The Virtual Room – Three.js

This project is a simple 3D virtual room built using Three.js.

## Features
- 3D scene with lighting
- Movable cube using keyboard controls
- Responsive canvas

## Controls
- Arrow keys to move the cube

## How it was built
GitHub Copilot was used to assist in generating the Three.js scene setup and controls.
